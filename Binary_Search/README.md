# 🔍 Binary Search

## The Mid Crisis 💀

The technique where one `mid` decides the fate of the entire search space. ✨

Left? Right? Equal?  
One wrong move and you're debugging for the next 30 minutes. 😭

Binary Search is about cutting down possibilities instead of checking everything one by one.

> Less searching. More eliminating. Maximum optimization.

## 💭 What I Learned
